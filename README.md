## Create Nextjs App

```shell
npx create-next-app@canary nextjs-starter \
  --use-pnpm --empty --turbo \
  --ts --eslint --tailwind --src-dir --app --import-alias "@/*"
```

## Run locally

```bash
pnpm dev
```
